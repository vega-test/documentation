A governance proposal located by either its ID or reference. If both are set, ID is used.


```graphql
proposal(
  id: ID
  reference: String
): Proposal
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>proposal.<b>id</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optionally, locate proposal by its ID
> 

#### [<code style={{ fontWeight: 'normal' }}>proposal.<b>reference</b></code>](#)<Bullet />[`String`](/testnet/api/graphql/scalars/string) <Badge class="secondary" text="scalar"/>
> Optionally, locate proposal by its reference. If ID is set, this parameter is ignored.
>
