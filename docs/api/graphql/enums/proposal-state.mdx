Various states a proposal can transition through:
Open -&#x003E;
- Passed -&#x003E; Enacted.
- Rejected.
Proposal can enter Failed state from any other state.


```graphql
enum ProposalState {
  STATE_FAILED
  STATE_OPEN
  STATE_PASSED
  STATE_DECLINED
  STATE_REJECTED
  STATE_ENACTED
  STATE_WAITING_FOR_NODE_VOTE
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_FAILED</b></code>](#) 
> Proposal became invalid and cannot be processed
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_OPEN</b></code>](#) 
> Proposal is open for voting
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_PASSED</b></code>](#) 
> Proposal has gained enough support to be executed
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_DECLINED</b></code>](#) 
> Proposal didn't get enough votes
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_REJECTED</b></code>](#) 
> Proposal could not gain enough support to be executed
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_ENACTED</b></code>](#) 
> Proposal has been executed and the changes under this proposal have now been applied
> 

#### [<code style={{ fontWeight: 'normal' }}>ProposalState.<b>STATE_WAITING_FOR_NODE_VOTE</b></code>](#) 
> Proposal is waiting for the node to run validation
>
