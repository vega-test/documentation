Subscribe to event data from the event bus


```graphql
busEvents(
  types: [BusEventType!]!
  marketId: ID
  partyId: ID
  batchSize: Int!
): [BusEvent!]
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>busEvents.<b>types</b></code>](#)<Bullet />[`[BusEventType!]!`](/testnet/api/graphql/enums/bus-event-type) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="enum"/>
> The types to subscribe to has to be an array
> 

#### [<code style={{ fontWeight: 'normal' }}>busEvents.<b>marketId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optional filter by market ID
> 

#### [<code style={{ fontWeight: 'normal' }}>busEvents.<b>partyId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optional filter by party ID
> 

#### [<code style={{ fontWeight: 'normal' }}>busEvents.<b>batchSize</b></code>](#)<Bullet />[`Int!`](/testnet/api/graphql/scalars/int) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> Specifies the size that the client will receive events in. Using 0 results in a variable batch size being sent. The stream will be closed if the client fails to read a batch within 5 seconds
>
