An order in the Vega network found by orderID


```graphql
orderByID(
  id: ID!
  version: Int
): Order!
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>orderByID.<b>id</b></code>](#)<Bullet />[`ID!`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> ID for an order
> 

#### [<code style={{ fontWeight: 'normal' }}>orderByID.<b>version</b></code>](#)<Bullet />[`Int`](/testnet/api/graphql/scalars/int) <Badge class="secondary" text="scalar"/>
> Version of the order (omitted or 0 for most recent; 1 for original; 2 for first amendment, etc)
> 

### Type

#### [`Order`](/testnet/api/graphql/objects/order) <Badge class="secondary" text="object"/>
