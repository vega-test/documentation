The interval for trade candles when subscribing via Vega GraphQL, default is I15M


```graphql
enum Interval {
  INTERVAL_BLOCK
  INTERVAL_I1M
  INTERVAL_I5M
  INTERVAL_I15M
  INTERVAL_I1H
  INTERVAL_I6H
  INTERVAL_I1D
}
```

#### [<code style={{ fontWeight: 'normal' }}>Interval.<b>INTERVAL_BLOCK</b></code>](#)
