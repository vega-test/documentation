A data source contains the data sent by a data source


```graphql
type Data {
  signers: [Signer!]
  data: [Property!]
  matchedSpecIds: [ID!]
  broadcastAt: Timestamp!
}
```

#### [<code style={{ fontWeight: 'normal' }}>Data.<b>matchedSpecIds</b></code>](#)<Bullet />[`[ID!]`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="list"/> <Badge class="secondary" text="scalar"/>
> List of all the data specs that matched this source data.
> When the array is empty, it means no data spec matched this source data.
> 

#### [<code style={{ fontWeight: 'normal' }}>Data.<b>broadcastAt</b></code>](#)<Bullet />[`Timestamp!`](/testnet/api/graphql/scalars/timestamp) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> RFC3339Nano formatted date and time for when the data was broadcast to the markets
> with a matching data spec.
> It has no value when the source data does not match any data spec.
>
