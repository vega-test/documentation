Subscribe to liquidity provisioning data


```graphql
liquidityProvisions(
  partyId: ID
  marketId: ID
): [LiquidityProvisionUpdate!]
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>liquidityProvisions.<b>partyId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> The party ID to subscribe for, empty if all
> 

#### [<code style={{ fontWeight: 'normal' }}>liquidityProvisions.<b>marketId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> The market ID to subscribe for, empty if all
> 

### Type

#### [`LiquidityProvisionUpdate`](/testnet/api/graphql/objects/liquidity-provision-update) <Badge class="secondary" text="object"/>
> The command to be sent to the chain for a liquidity provision submission
