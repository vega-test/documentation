Various proposal types that are supported by Vega


```graphql
enum ProposalType {
  TYPE_NEW_MARKET
  TYPE_UPDATE_MARKET
  TYPE_NETWORK_PARAMETERS
  TYPE_NEW_ASSET
  TYPE_NEW_FREE_FORM
  TYPE_UPDATE_ASSET
}
```
